# 📘 Deep Learning from Scratch Study (Vol. 1)
> **"Understanding the internal mechanics of Neural Networks from the ground up."**

이 저장소는 《밑바닥부터 시작하는 딥러닝 1》을 공부하며 라이브러리(PyTorch, TensorFlow)를 사용하지 않고 오직 **NumPy**만을 활용해 딥러닝의 기초를 구현한 기록입니다. 

## 🎯 학습 목표 (Learning Objectives)
- 인공 신경망의 수학적 원리(오차역전파, 손실 함수 등)를 밑바닥부터 구현하며 이해합니다.
- 효율적인 수치 계산을 위한 NumPy 벡터화 연산 능력을 배양합니다.
- 복잡한 AI 모델을 모듈화하여 설계하는 엔지니어링 역량을 키웁니다.
